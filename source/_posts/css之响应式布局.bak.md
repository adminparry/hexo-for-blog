### 左边固定右边自适应

``` bash
<div class="left"></div>
<div class="right"></div>
<style>
	.left{
		width: 200px;float: left;background: red;
	}
	.right{
		overflow: hidden;background: green;
	}
</style>
```

More info: [Writing](https://hexo.io/docs/writing.html)
